# Recon Diver static website

This folder is a cleaned static export of the original legacy site.

## Structure

- `index.html` main homepage
- `about/`, `instructors/`, `map/`, `pool-service/`, `schedule/`, `videos/` top-level pages
- `articles/` article index and article detail pages
- `courses/` all course pages grouped by learning level
- `trips/` trip pages
- `assets/` project images, CSS, fonts, and front-facing assets
- `vendor/` legacy libraries kept only for compatibility with the static export
