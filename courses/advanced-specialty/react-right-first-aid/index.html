                                                                          React Right & First Aid (ปฐมพยาบาล)


                                                                          หลักสูตรการปฐมพยาบาล และช่วยชีวิตฉุกเฉิน React Right & First Aid 
                                                                          เราคิดค่าใช้จ่ายท่านละ                                                                                                                                                     4,000  บาท 

                                                                          • รวม  ค่าใบอนุญาตของสถาบัน SSI / React Right & First Aid e-Card 
                                                                          • รวม  ค่าอุปกรณ์การฝึกสอน
                                                                          • รวม  ค่าตำราเรียนแบบ Online e-Learning ของ SSI และภาคปฏิบัติ Onsite แล้ว
                                                                          • หลักสูตรนี้ไม่มีภาคสระ และภาคทะเล

                                                                           

                                                                          รายละเอียดของหลักสูตร

                                                                          ยกระดับทักษะการกู้ชีพมาตรฐานสากลกับหลักสูตร SSI React Right

                                                                          เนื้อหาครอบคลุมทักษะการช่วยชีวิตขั้นพื้นฐาน (Primary Assessment), การทำ CPR ที่มีประสิทธิภาพ, การใช้เครื่องกระตุ้นหัวใจ (AED) ไปจนถึงการบริหารจัดการออกซิเจนในสถานการณ์ต่างๆ หลักสูตรนี้ไม่ได้เพียงแค่สอนให้คุณ 'รู้' แต่ฝึกให้คุณ 'ทำเป็น' เพื่อสร้างความปลอดภัยให้แก่คนรอบข้างและเพื่อนนักดำน้ำในทุกสถานการณ์

                                                                          เพราะทุกวินาทีมีค่าเมื่อเกิดเหตุฉุกเฉิน หลักสูตร SSI React Right จะเปลี่ยนคุณให้พร้อมรับมือกับวิกฤตการณ์ทางการแพทย์ได้อย่างไม่คาดคิด
